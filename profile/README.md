# 🌊 Shinsekai

Shinsekai는 Java 기반 백엔드와 Next.js 프론트엔드를 활용하여 **풀스택 웹 서비스를 기획, 개발, 배포**하는 팀입니다.  
우리는 실전과 같은 환경에서 프로젝트를 운영하며, 팀원들의 기술 성장과 협업 능력을 함께 키워가고 있습니다.

## 🛠️ 사용 기술 스택

### Backend
- Java, Spring Boot, JPA, QueryDSL, Redis, MySQL
- Spring Security, JWT, Swagger, Gradle

### Frontend
- Next.js, React, TypeScript
- Tailwind CSS, Zustand, SWR, pnpm

### DevOps & Infra
- AWS EC2, S3, Route53, GitHub Actions
- Docker, Nginx, Let's Encrypt (SSL), Ubuntu

# 🌟 Spharos – 신세계 스타벅스 리빌딩 프로젝트

> **Spharos**는 사용자의 편의성을 고려한 UI/UX와 강력한 백엔드 설계를 바탕으로, 누구나 쉽고 빠르게 상품을 탐색하고 구매할 수 있는 쇼핑 플랫폼입니다.

- 실시간 재입고 알림, 사용자 맞춤 추천
- 다양한 카테고리 필터링, 직관적인 상품 상세 정보
- 관리자/사용자 기능 분리와 보안 강화
- AWS 기반의 안정적인 배포 환경
